Sobre mí

Me llamo Marcos y soy estudiante de desarrollo web. Desde que empecé a aprender HTML y CSS me he interesado por entender cómo se construyen las páginas web desde la base, cuidando tanto la estructura como el diseño visual.

Me gusta trabajar con código limpio, bien organizado y fácil de mantener. Considero que una buena base en HTML semántico y CSS bien estructurado es fundamental antes de pasar a tecnologías más avanzadas.

Actualmente sigo formándome y realizando proyectos personales para mejorar mis habilidades y adquirir una base sólida en desarrollo front-end.

Habilidades principales

  • Maquetación web con HTML5 semántico
  • Estilos CSS organizados y reutilizables
  • Diseño responsive básico
  • Buenas prácticas de accesibilidad
  • Organización de proyectos web

Proyectos

Desarrollo de una página web personal tipo landing page, utilizando únicamente HTML y CSS. El objetivo principal fue crear una estructura clara y un diseño visual coherente.
