# Portfolio Profesional — Marcos Heredia

¡Bienvenido! Esta es la versión pública y personalizable del portfolio profesional de Marcos Heredia.

Presenta una landing estática, desarrollada únicamente con HTML y CSS (sin JavaScript), pensada para mostrar tu perfil, habilidades y proyectos de forma clara, rápida y accesible.

Demo rápida
- Abre este link en tu navegador: portfolio-rho-beige-54.vercel.app
- Abre `index.html` en tu navegador para ver el sitio localmente.

Características principales
- Diseño responsivo con CSS Grid y Flexbox.
- Interfaz limpia y profesional, sin dependencias externas.
- Formularios HTML listos para integrarse con servicios de correo (p. ej. Formsubmit).

Estructura del repositorio

- `index.html` — página principal y contenido.
- `css/style.css` — estilos y comentarios explicativos.
- `img/` — imágenes y recursos usados en la página.

Objetivo

Crear una presentación profesional y accesible que permita a reclutadores y colegas revisar rápidamente tu experiencia y proyectos, manteniendo el código simple y fácil de adaptar.

Cómo usar

1. Clona este repositorio o descarga los archivos.
2. Abre `index.html` en tu navegador (doble clic o `Ctrl+O`).
3. Personaliza textos, enlaces e imágenes en `index.html`.
4. (Opcional) Subir a un hosting gratuito: Vercel, Netlify o GitHub Pages.

Consejos rápidos para personalizar

- Sustituye `img/Mhsr.jpg` por tu foto profesional.
- Actualiza la sección “Proyectos” con enlaces reales y capturas.
- Añade un CV en PDF y enlázalo desde la sección “Sobre mí”.
- Si quieres recibir mensajes desde el formulario sin backend, usa Formsubmit o Formspree y sigue su proceso de activación.

Requisitos y buenas prácticas

- Estructura semántica: usa `header`, `nav`, `main`, `section`, `footer`.
- Accesibilidad: añade `alt` descriptivos a las imágenes y asegura contraste suficiente.
- CSS separado: todo el CSS reside en `css/style.css` con comentarios para facilitar mantenimiento.

Despliegue recomendado

- GitHub Pages: ideal para repositorios estáticos.
- Vercel o Netlify: despliegue automático desde GitHub y soporte para formularios (Netlify Forms) si prefieres no usar servicios externos.
