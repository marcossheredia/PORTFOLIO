Sobre mí

Soy desarrollador FullStack con experiencia en la creación de aplicaciones web modernas, eficientes y centradas en el usuario. Trabajo habitualmente con Vue 3, JavaScript y CSS moderno, cuidando tanto la arquitectura del código como el diseño visual y la usabilidad.

Me gusta construir interfaces limpias, mantenibles y bien estructuradas, prestando especial atención al rendimiento, la experiencia en dispositivos móviles y las buenas prácticas de desarrollo. Tengo experiencia desarrollando proyectos reales completos, desde paneles de administración hasta aplicaciones orientadas a negocio, incluyendo gestión de productos, pedidos y contenido.

Busco seguir creciendo como desarrollador dentro de equipos donde se valore el código bien hecho, la mejora continua y el enfoque práctico para resolver problemas reales.
